What is Web 3.0?
Web 3.0 is evolution from the read and write web 2.0 to the read, write and execute. web 3.0 supposed to be more quality and have better back-end.


Will it be the next trend?
In my opinion, Web 3.0 will be the next trend, because web 3.0 is a huge advance from the web 2.0, where it can be very inteligent and very interactive with the user. Web 3.0 will know to how to handle alot of complex data, and decide which data is best relvant for the user.
Find 3 websites which are web 3.0
  1. Facebook - This website handle alot of data and shows the users advertisment, articles, friend's status, friend's images and more. The site decides which data to show and which not.
  2. Google - This website not just do input-ouput operations, on search query google returns results acording the specific user data, such location, history searches, placed he had visited and more. To two users who searches the same thing, different results will be shown.
  3. Youtube - This website not simply shows videos, on every video there will be a bar with relevent videos. On the main page there will be recommended videos according the history watch\search.
